# יומן מסחר — Trading Journal

אפליקציית Web מקומית לתיעוד וניתוח עסקאות מסחר בשוק ההון.

## הפעלה

פתח את הקובץ `index.html` ישירות בדפדפן (Chrome / Edge / Firefox).  
אין צורך בשרת, התקנות, או חיבור אינטרנט (מלבד טעינה ראשונה של פונטים ו-Chart.js).

## חדרים

| חדר | תיאור |
|-----|-------|
| יומן עסקאות | הוספה, עריכה, מחיקה וסינון של כל העסקאות |
| תיק מסחר | תצוגת תיק חי עם גרף עוגה ו-P&L לא ממומש |
| לקחים חודשיים | ניתוח אוטומטי + לקחים אישיים מסודרים לפי חודש |

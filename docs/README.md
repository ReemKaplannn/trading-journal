### 06-ENGINEER-MODE.pdf
**מוד: מהנדס תוכנה בכיר (Staff Engineer)**

מגדיר כיצד Claude פועל כ-Staff Engineer ברמת Google / Meta / Stripe.
מכיל:
- פילוסופיית קוד: Production-ready מהיום הראשון, No dead code, Readability > Cleverness

- ארכיטקטורת SwingEdge: folder structure, state management, TradingView widgets
- React component template + Hooks pattern
- Performance best practices (memo, lazy load, IntersectionObserver)
- Quality checklist לפני commit
- Security best practices
- טבלת Tech Decisions של SwingEdge (Vite, Tailwind, Supabase, וכו')
